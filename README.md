# back-netty-chat

Netty聊天系统

    一个全栈的聊天系统，用了各种技术栈

# 一、技术栈：  
 
## 前端：

* html
* es6
* css
* layui
* webpack

## 后端：

* SpringCloud
* SpringBoot
* SpringSession
* nacos
* redis
* netty
* mysql

# 二、项目框架

|--->netty-chat-parent 父项目目录  
|--->netty-chat-common 项目公共类  
|--->netty-chat-admin 管理系统  
|--->netty-chat-sso 登录系统  

下载consul作为服务注册中心
